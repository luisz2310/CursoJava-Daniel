# 📘 Clase 1 — Variables en Java

Este ejemplo explica cómo usar variables en Java de forma básica.

## 🧠 ¿Qué es una variable?
Una variable es un espacio en memoria donde guardamos un valor.

## ✏️ Ejemplos del código

```java
byte m;      // Declaración (crear variable)
m = 2;       // Asignación (dar valor)

byte n1 = 4; // Declarar e inicializar en una sola línea

char letra = 'a'; // Variable tipo carácter
